# Don't Fear The Reaper

A Subnautica mod that tweaks the behavior, aggression, and presence of the Reaper Leviathan to create a more balanced and customizable gameplay experience.

## Features
- Adjust the spawn rate of Reaper Leviathans
- Modify aggression, speed, and roar behavior
- Customize visual and audio cues
- Localized for multiple languages
- Modular design for easy expansion

## Installation
1. Download and install [BepInEx](https://www.nexusmods.com/subnautica/mods/1108) and [Nautilus](https://www.nexusmods.com/subnautica/mods/1262) for Subnautica.
2. Download the latest release of Dont Fear The Reaper from the [Releases](https://github.com/visolar6/Dont-Fear-The-Reaper/releases) page or build from source.
3. Extract the `DontFearTheReaper` folder into your `Subnautica/BepInEx/plugins` directory.
4. Launch Subnautica. The mod will load automatically.

## Configuration
- Edit the `Options` in-game or modify the `Localizations.xml` for language support.
- Advanced users can tweak values in the mod's config files for further customization.

## Development
- Built for Subnautica using BepInEx, Nautilus, and Harmony.
- Main code in `DontFearTheReaper/`.
- Patches and utilities in `Patches/` and `Utilities/`.
- See `DontFearTheReaper.csproj` for dependencies and references.
